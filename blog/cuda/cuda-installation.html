Installing CUDA on Ubuntu 22.04

This guide walks you through the steps to install CUDA on Ubuntu 22.04, enabling GPU acceleration for tasks like deep learning and high-performance computing.

1. Introduction

We will install the CUDA toolkit, set up our environment, and verify the installation. Having CUDA properly installed is essential for leveraging the computational power of NVIDIA GPUs.

2. Prerequisites

  • An NVIDIA GPU that supports CUDA.
  • Ubuntu 22.04 installed on your system.
  • Basic knowledge of the Linux command line.

3. Step 1: Access the CUDA Download Page

Visit the official CUDA Downloads Page and select the correct options for your OS, architecture, and version.

4. Step 2: Download the Installation Package

Follow the instructions on the NVIDIA website to download the `.deb` package. You can use the `wget` command to download it directly from the terminal.

5. Step 3: Install the CUDA Toolkit

  1. Update the package list: 
    sudo apt-get update
  2. Install the downloaded package: 
    sudo dpkg -i cuda-repo-ubuntu2204-X.Y-local_X.Y.Z-1_amd64.deb

    Replace X.Y and X.Y.Z with the version numbers from the download link.

  3. Install CUDA: 
    sudo apt-get install cuda

6. Step 4: Verify the Installation

Run the following command to verify the CUDA installation:

nvcc --version

You should see the version of the CUDA compiler if the installation was successful.
